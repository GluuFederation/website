  • Webinar: How 11 Universities use Gluu EDU to Support a Shibboleth IdP

    Gluu is an InCommon Affiliated and a sponsor of the Shibboleth foundation. In this webinar, Gluu CEO Mike Schwartz will demonstrate how the Gluu EDU Server can simplify certificate management and provide an easy framework for creating Shibboleth/SAML SSO with InCommon sponsored partners.

  • Strong Authentication

    Demo how to use OX open source for strong, multi-step, multi-factor authentication. Demo includes SAML SSO to Salesforce, and OAuth 2.0 OpenID Connect authentication. Specifically show mobile 2-factor authentic services of Duo and OpenID. Also shows how Jython is used to access JSON/REST external authentication services.

  • Gluu Multi-Party SAML Federation Demo: OX Open Source

    Gluu CEO Michael Schwartz demos how to use the Gluu SAML platform to create SSO with websites using SAML multi-party federation metadata. He goes on to give an explanation of existing Shibboleth multi-party federations in the Higher Education domain, including an in depth discussion of the tools and rules that make up InCommon. The video concludes with a second demo of how to use the Gluu platform to host a local federation.

  • OX UMA Policy Management

    An overview of how OX implements UMA for Web Access Management (WAM) and how an administrator at an organization can use the OX admin GUI to create a resource set to define which URL's to protect; then how to define the policies that define the business logic for protection.

  • Demo of UMA for OAuth 2.0 Web Access Management

    In this demo, Yuriy Z. gives an overview of the Java code for an open source application he wrote to demonstrate how an UMA client could request authorization from an UMA resource server, assuming the person uses OpenID Connect for authentication.

  • SAML / OpenID Connect SSO Demo

    Quick demo showing strong authentication and SSO between a SAML site (Saleforce) and an OpenID Connect site. Also shows Gluu Trust Relationship configuration needed to support Salesforce for SSO.

  • SAML Active Directory (LDAP) Mapping in the Gluu OX platform

    Mike shows how to map people from an organization's existing LDAP server, and how to use Java or Python code to transform attribute values or even create new attributes. Attribute mapping increases the performance of queries from the IDP for faster SSO, and enables one mapping point for both SAML and OpenID Connect.
